ONSTS Favicon Setup Instructions
=====================================

To complete the favicon setup, you need to add these files to the public directory:

Required Files:
1. favicon.ico (16x16, 32x32, 48x48 - ICO format)
2. favicon-16x16.png (16x16 PNG)
3. favicon-32x32.png (32x32 PNG)  
4. apple-touch-icon.png (180x180 PNG for iOS)

How to Create:
1. Use your ONSTS logo image
2. Resize to the required dimensions
3. Convert to appropriate formats
4. Place in this public directory

Online Tools:
- favicon.io - Easy favicon generator
- realfavicongenerator.net - Comprehensive favicon generator
- canva.com - For resizing images

Once you add the files, the favicon will appear in browser tabs automatically.
