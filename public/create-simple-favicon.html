Instructions:

  1. Click "Download Favicon" to get a simple ONSTS favicon
  2. Rename the downloaded file to match the required names:
    • favicon-32x32.png
    • favicon-16x16.png (resize to 16x16)
    • apple-touch-icon.png (resize to 180x180)
  3. Convert one copy to favicon.ico format
  4. Place all files in the public directory
